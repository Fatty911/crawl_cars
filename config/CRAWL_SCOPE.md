# 爬取范围

## 当前目标车型

只保留轿车、跑车和 SUV：

- 轿车相关级别：微型车、小型车、紧凑型车、中型车、中大型车、大型车
- 跑车
- SUV

## 当前排除车型

以下类型不进入最终数据；如果懂车帝列表接口直接返回明确级别，也会在列表阶段跳过：

- MPV
- 房车
- 货车、卡车、微卡、轻卡
- 皮卡
- 轻客、客车、微面、面包车
- 厢式、载货、牵引、自卸等商用车关键词

## 链接范围记录

当前爬虫入口仍使用汽车之家按品牌字母页和懂车帝车系列表接口。

汽车之家的字母页会混合返回多种级别，列表链接本身没有稳定体现 MPV、房车、货车等排除范围；汽车之家车系页历史上可通过 `levelname` / `levelid` 识别车型级别，但当前爬虫不额外请求车系首页，只在配置数据生成阶段按“级别/车身结构”等字段过滤。

懂车帝列表接口如果返回 `level_name`、`series_level_name`、`car_level_name`、`type_name`、`category_name`、`rank_name` 等明确级别字段，会在车系列表阶段跳过排除类型；否则会继续下载并在解析后的配置字段中二次过滤。
