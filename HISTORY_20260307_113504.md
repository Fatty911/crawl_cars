### 问题：GitHub Actions 免费额度不够

**现象**：报错 "The job was not started because recent account payments have failed or your spending limit needs to be increased"

**原因**：GitHub Actions 免费版每月 2000 分钟
- 每次 2 小时 = 120 分钟
- 每天 2 次 = 240 分钟
- 一个月 = 7200 分钟
- 严重超标！

**解决方案**：

| 平台 | 限制 | 适合爬虫 | 推荐 |
|------|------|----------|------|
| GitHub Actions | 2000分钟/月 | ❌ 不够 | - |
| VPS | 无限制 | ✅ 最佳 | ⭐⭐⭐⭐⭐ |
| Railway | $5/月额度 | ✅ 可行 | ⭐⭐⭐⭐ |
| Vercel/Zeabur | 有超时限制 | ❌ 不适合 | - |

### 新增功能

#### 1. 代理管理器 (proxy_manager.py)

**功能**：
- 支持解析 V2Ray 订阅链接
- 支持解析 Clash 配置文件
- 支持手动添加 HTTP/SOCKS5 代理
- 负载均衡策略：
  - random: 随机选择
  - round_robin: 轮询
  - least_used: 最少使用
  - best_performance: 最佳性能

**使用方法**：
```bash
# 解析订阅
python3 proxy_manager.py --sub "https://订阅链接"

# 测试代理
python3 proxy_manager.py --test

# 查看列表
python3 proxy_manager.py --list
```

**注意**：
- SS/VMess/Trojan 需要 Clash/V2Ray 本地客户端转换为 HTTP 代理
- 或使用 privoxy 将 SOCKS5 转为 HTTP
- 直接使用 HTTP/SOCKS5 代理最简单

#### 2. VPS 一键部署脚本 (deploy_vps.sh)

**功能**：
- 自动安装依赖
- 创建定时任务（每天 2:00 和 14:00）
- 随机延迟启动
- 自动 commit/push 进度

**使用方法**：
```bash
curl -fsSL https://raw.githubusercontent.com/Fatty911/test_crawl/main/deploy_vps.sh | bash
```
